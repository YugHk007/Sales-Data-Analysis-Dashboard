# Sales Data Analysis Dashboard

## Project Overview
This project analyzes sales data from the Superstore dataset and provides interactive visualizations for business insights.

## Features
- Sales Analysis
- Profit Analysis
- Regional Performance
- Category-wise Performance
- Interactive Dashboard
- Data Visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Plotly
- Matplotlib

## Dataset
Sample Superstore Dataset
